## Используемые технологии
- PHP (валидация, PDO, подготовленные запросы)
- MySQL (база данных `u82293`)
- HTML/CSS (адаптивная форма)

## Структура базы данных
- `users` – основные данные пользователя
- `programming_languages` – справочник языков программирования
- `user_languages` – связь пользователей с выбранными языками (многие ко многим)

## Как развернуть у себя
1. Импортировать `schema.sql` в вашу БД
2. Настроить параметры подключения в `index.php` (хост, логин, пароль)
3. Открыть `index.php` в браузере

## Валидация полей (все поля проверяются на сервере)
- **ФИО**: только буквы, пробелы, дефис, не длиннее 150 символов
- **Телефон**: корректный формат (цифры, +, -, пробелы, скобки)
- **Email**: валидный email-адрес
- **Дата рождения**: реальная дата, возраст не больше 120 лет
- **Пол**: один из трёх вариантов (male/female/other)
- **Языки программирования**: минимум один из разрешённого списка
- **Биография**: не более 5000 символов
- **Контракт**: обязательное согласие
